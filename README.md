# Inteligência Artificial — AI Chess Assistant

Projeto desenvolvido no âmbito da unidade curricular de **Inteligência Artificial**, focado na aplicação de técnicas de visão computacional para análise automática de tabuleiros de xadrez.

## Objetivo
Desenvolver uma aplicação capaz de:
- Detetar automaticamente peças de xadrez a partir de imagens
- Reconstruir o estado do tabuleiro em notação FEN
- Integrar com o motor Stockfish para sugestão de jogadas

## Pipeline do sistema
- Geração e etiquetagem de imagens
- Treino de modelos YOLOv8
- Avaliação e comparação de modelos
- Inferência em tempo real
- Integração com Stockfish

## Tecnologias
- Python
- YOLOv8 (Ultralytics)
- Roboflow
- Stockfish
- OpenCV / mss
- Tkinter

## Contexto académico
Projeto académico desenvolvido no ano letivo 2024/2025.
